# REC Brand Assets

Mockups PNG de los templates de marca de Rivas Esports Club (REC) v1.1.

3 familias:
- 01-institucional: portada, slide, square
- 02-deportiva: ficha, slide, story, square, reel
- 03-difusion: portada, slide, story, square

Generado con `render-families.mjs` desde el repo privado `rec-vault`.

Solo se hostea aqui para que Canva pueda importar los PNGs via URL publica.
